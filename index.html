História dos computadores

Os computadores foram criados para geração de cálculos complexos, criptgrofia e criptoanálises muitos utilizados em épocas de guerra, principalmente na segunda guerra mundial tivemos os de 1 geração como mark, colossus e Eniac, tivemos a 2 geração (1959 a 1965) marcada pela uso de transistores, 3 Geração (1965 a 1970) marcada pela utilização dos chips 4 geração (a partir de 1971) com evolução de pesquisa os computadores se tornaram menores e com preços mais acessíveis a população, a 5 Geração(dias Atuais) se da partir dos dias atuais como a criação de criptomoedas, IA, Metaverso.

O que são Servers?

O que são servidores de acordo com coodesh.com/

um servidor é um computador ou software mais potente que um desktop comum. Ele é capaz de conter um ou mais processadores, além de bancos de memórias, portas de comunicação, diferentes softwares e sistemas de armazenamento de dados.

Servidores também tem sistemas operacionais específicos, e também executam códigos de comandos backend como php, java, entre outras linguagens, o server que recebe as requisições http feitas pelo lado do cliente.

Como é uma estrutura básica de um HTML?

A estrutura básica de um documento html inclui as seguintes tags !DOCTYPE htmlpara definir a versão do html, temos html lan pra definir a linguagem headhead para conter informação de metadados e links de script temos a tag meta charset para definir o paramêtro de codificação temos a tag title para definir o título do site, a tag body para definir o conteúdo principal do site como imagens, links, e entre outros sublinhado quando necessário.

Entendendo a diferença entre listas ordenadas e não ordenadas

Lista ordenadas é uma lista desenvolvida através da tag olcriando uma sequência a cada item por tag li a listas não ordenadas são desenvolvidas através da tag ul.

Exemplos

  1. lista ordenada: item 1
  2. lista ordenada: item 2
  3. lista ordenada: item 3
